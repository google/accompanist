LoadPainter

class LoadPainter<R> : Painter, RememberObserver
Content copied to clipboard

A generic image loading painter, which provides the Loader interface for image loading libraries to implement. Apps shouldn't generally use this function, instead preferring one of the extension libraries which build upon this, such as the Coil and Glide libraries.

Instances can be created and remembered via the rememberLoadPainter function.
